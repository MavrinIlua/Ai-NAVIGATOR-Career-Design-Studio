## 🧭 1. Архитектура потока данных и управления состоянием

Приложение построено по модульной структуре Single Page Application (SPA). Главным источником истины является корневой компонент `App.tsx`.

```
                   ┌───────────────────────────────────────┐
                   │                App.tsx                │
                   │(Режим, API-ключ, Элементы, Превью)   │
                   └───────────────────┬───────────────────┘
                                       │
        ┌──────────────┬───────────────┼───────────────┬──────────────┐
        ▼              ▼               ▼               ▼              ▼
┌──────────────┐┌──────────────┐┌──────────────┐┌──────────────┐┌──────────────┐
│  Header.tsx  ││CareerExplorer││ImageGenerator││ImageEditor.tsx││ Gallery.tsx  │
│ (Nav & Tabs) ││ (AI Career)  ││ (Text->Media)││ (Inpainting) ││ (Portfolio)  │
└──────────────┘└──────────────┘└──────────────┘└──────────────┘└──────────────┘
```

### Основные состояния (States) в `App.tsx`:
- `mode` (`AppMode`): Текущий экран приложения.
- `hasApiKey` (`boolean`): Логический флаг, блокирующий приложение до подтверждения валидности API-ключа (критично для работы со сложными моделями).
- `items` (`GeneratedMedia[]`): Хранилище сгенерированных карточек профессий, картинок и видео. Синхронизировано с `localStorage` по ключу `nav_items_v3`.
- `latestItem` (`GeneratedMedia | null`): Элемент, который в данный момент отображается в правой закрепленной колонке предварительного просмотра (Preview Sidebar).

## 📡 2. Настройка и интеграция Google GenAI SDK

Для работы приложения необходим пакет `@google/genai` (версия `^1.38.0` или выше). Вся работа с API инкапсулирована в `geminiService.ts`.

### Безопасная инициализация клиента
Чтобы избежать падения приложения в контейнере при отсутствии ключа на этапе сборки, инициализация клиента вынесена в ленивую функцию:

```typescript
import { GoogleGenAI } from '@google/genai';

let ai: GoogleGenAI | null = null;

export function getGeminiClient(): GoogleGenAI {
  if (!ai) {
    // 1. Попытка взять ключ из глобальных переменных сборщика
    // 2. Или из хранящегося в памяти временного хранилища
    const apiKey = process.env.GEMINI_API_KEY || (window as any)._temporaryGeminiKey;
    if (!apiKey) {
      throw new Error("API-ключ Google GenAI отсутствует. Настройте .env или выберите ключ в интерфейсе.");
    }
    ai = new GoogleGenAI({ apiKey });
  }
  return ai;
}
```

### Модели и их роли:
1. **`gemini-2.5-flash`** — используется для генерации структурированных карьерных планов в текстовом формате. Благодаря высокой скорости и низкой стоимости, она идеально подходит для интерактивных диалогов и профориентационного тестирования.
2. **`imagen-3.0-generate-002`** — используется для создания иллюстраций и концепт-артов в модуле ImageGenerator.
3. **`veo-2.0-generate-001`** — используется для создания учебных синематиков и коротких динамических видеороликов (симуляций рабочей среды).

## 🎨 3. Стилизация и пользовательский опыт (UX/UI)

Проект использует строго академический, но современный ИТ-стиль, вдохновленный дизайном экосистемы МГПУ:
- **Базовый фон:** Спокойный ультра-светлый серый (`#FDFDFD`) для уменьшения нагрузки на глаза школьников при длительном чтении.
- **Главные акценты:**
  - Глубокий красный (`bg-red-600`, `text-red-600`) — символизирует академическую приверженность МГПУ и лидерство.
  - Насыщенный теплый янтарно-желтый (`text-amber-500`) — для элементов видео и вспомогательных виджетов.
- **Оформление карточек:** Закругленные края в стиле Apple (`rounded-3xl` и `rounded-[2.5rem]`), деликатные тени (`shadow-2xl shadow-gray-200/50`) для создания эффекта легкости и чистоты интерфейса.

## 🛡️ 4. Гарантии стабильности и обработки ошибок

Если ИИ-агент или разработчик переносит этот проект на сторонний хостинг, убедитесь в соблюдении следующих условий для исключения сбоев:

1. **Обработка истечения Blob-ссылок (Object URLs):**
   Сгенерированные в браузере ссылки на видео (Blob URLs) инвалидируются при перезагрузке страницы. В `App.tsx` встроен фильтр в конструктор начального состояния, который обнуляет истекшие ссылки:
   ```typescript
   const saved = localStorage.getItem('nav_items_v3');
   const parsed = saved ? JSON.parse(saved) : [];
   return parsed.map(item => {
     if (item.type === 'video' && item.url.startsWith('blob:')) {
       return { ...item, url: '' }; // маркер истечения срока ссылки
     }
     return item;
   });
   ```

2. **CORS и Смешанный контент:**
   При генерации изображений через API Google возвращает массив байтов (base64) либо прямые подписанные ссылки. Конвертируйте base64 в локальные Blob-объекты для гарантии автономной работы без CORS-ограничений целевых серверов Google Cloud Storage.

3. **Адаптивность (Responsive):**
   Интерфейс распределен по сетке Tailwind: `grid grid-cols-1 lg:grid-cols-12 gap-10`. 
   Левая колонка (`lg:col-span-6`) — интерактивный рабочий пульт. Правая колонка (`lg:col-span-6`) — закрепленное превью с плавающим поведением `sticky top-28`. Это обеспечивает идеальный обзор результатов как на Full-HD мониторах учебных классов, так и на мобильных устройствах.
