# AI Career Navigator Studio

**Интерактивный программный комплекс цифровой профессиональной навигации и профориентации для учащихся IT-классов и студентов.**

Разработано в рамках выпускной квалификационной работы (ВКР) магистратуры **ГАОУ ВО МГПУ** (Московский городской педагогический университет) по направлению педагогического образования и методики обучения информатике.

---

## 📌 О проекте

**AI Career Navigator Studio** — это веб-приложение (SPA), разработанное на базе современных веб-технологий (React, TypeScript, Vite, Tailwind CSS), которое выступает в роли интерактивного симулятора профессиональной среды в сфере искусственного интеллекта (Artificial Intelligence).

Приложение разработано для практического подтверждения гипотез научно-методической работы на тему:
> **«Методика применения ИИ-инструментов в профориентационной работе с учащимися ИТ-классов»**  
> (или *«Система профессиональной навигации для учащихся ИТ-классов в профессиях ИИ-направлений»*)

### 🎯 Педагогические цели комплекса:
1. **Деятельностная проба пера:** Обучающиеся не просто читают описания профессий, а взаимодействуют с реальными инструментами ИИ, выполняя роли специалистов.
2. **Формирование представлений о новых ИИ-профессиях:** Ознакомление со сквозными специальностями (ML-инженер, разработчик AI-агентов, специалист по RAG-системам, ИИ-этик и др.), требования к которым еще не устоялись в традиционных классификаторах.
3. **Развитие гибких навыков (Soft Skills):** Навыки промпт-инжиниринга, критической оценки генеративных результатов, этики использования нейросетей и системного мышления.

---

## 🛠️ Функциональные модули приложения

1. **Career Explorer (Исследователь профессий):**
   - Интерактивный интерактивный каталог ИИ-специализаций (более 20 направлений — от инфраструктурных инженеров до гуманитарных ИИ-специалистов).
   - Динамическая генерация паспорта профессии (компетенции, ежедневные задачи, образовательный трек и учебный промпт) на базе модели `gemini-2.5-flash`.

2. **Image Generator (Генератор медиа):**
   - Интеграция с моделями Imagen 3 и Veo для создания изображений и анимаций, иллюстрирующих будущую профессиональную среду.

3. **Image Editor (Редактор концептов):**
   - Инструмент редактирования и локальной модификации сгенерированных графических материалов (симуляция задач AI-дизайнера / AI Creator).

4. **Gallery (Портфолио результатов):**
   - Хранилище сгенерированных профилей профессий и связанных с ними медиаматериалов с локальным кэшированием (собирает локальное портфолио учащегося).

## 🚀 Как запустить проект локально

### Требования
- Установленный **Node.js** (версия 18.x или выше)
- Свободный порт `3000` (настроен по умолчанию)

### Шаги установки

1. **Клонирование репозитория и установка зависимостей:**
   ```bash
   npm install
   ```

2. **Настройка API ключа:**
   Создайте файл `.env` в корневой директории проекта и добавьте ваш ключ доступа Google Gemini API:
   ```env
   GEMINI_API_KEY=ваш_api_ключ_здесь
   ```
   *Примечание: Вы также можете использовать встроенный графический интерфейс выбора ключа в самом приложении.*

3. **Запуск в режиме разработки:**
   ```bash
   npm run dev
   ```
   Проект будет запущен по адресу: [http://localhost:3000](http://localhost:3000)

4. **Production сборка проекта:**
   ```bash
   npm run build
   ```

## 📂 Структура исходного кода

- `App.tsx` — Главный координатор приложения, обрабатывает сессии, API-ключи и роутинг режимов.
- `/components` — Папка интерактивных UI компонентов:
  - `Header.tsx` — Навигационный бар с акцентной темой.
  - `CareerExplorer.tsx` — Движок интерактивного тестирования и симуляции карьерных паспортов.
  - `ImageGenerator.tsx` — Модуль ввода промптов и генерации медиа (изображений и видео).
  - `ImageEditor.tsx` — Встроенный умный графический ИИ-редактор.
  - `Gallery.tsx` — Портфолио выполненных работ учеников.
- `geminiService.ts` — Изолированный сервис для отправки запросов к Google GenAI SDK.
- `constants.tsx` — Список из 20+ направлений ИИ-профессий и начальные конфигурации стилей.
- `types.ts` — Строгие TypeScript интерфейсы и перечисления состояний.
- `CONCEPT_AND_BLUEPRINT.md` — Фундаментальный научно-методический документ для дипломного совета МГПУ и дорожная карта дальнейшей разработки.
- `METHODOLOGY_GUIDE.md` — Методическое руководство для учителей информатики и вузовских преподавателей по проведению профессиональных проб с использованием комплекса.
